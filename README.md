# ArcadeHub

A full-stack gaming portal built using React, Node.js, Express, and MongoDB.

## Features

- User Authentication using JWT
- Protected Routes
- Multiple Browser Games
- Responsive Frontend
- Backend API Integration
- MongoDB Database Support

## Games Included

- Tic Tac Toe
- Sudoku
- Minesweeper
- Rock Paper Scissors
- Dice Game
- 2048
- Tower of Hanoi
- Hand Cricket

## Tech Stack

### Frontend
- React
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT
- bcrypt
- Cookies

## Installation

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd my-app
npm install
npm run dev
```
